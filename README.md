## DockFlow 是什么

DockFlow 是一款 macOS 专属工具，住在菜单栏里，安安静静，不占屏幕空间。它的核心逻辑很简单——让你把不同场景下需要用到的应用、文件夹、甚至网址链接，提前配置好保存成一套「方案」，用的时候一键切换过去。

开发者说他做这个软件，是因为自己作为程序员在 Java、PHP、设计稿之间来回切换，每次都要花时间重新整理 Dock，实在太烦了。于是干脆自己写一个。

这种「自己用不爽所以自己写」的软件，往往都挺好用的。

## 主要功能

**创建图标方案**

你可以给每种工作模式配置专属的 Dock 布局。比如「开发模式」只留 Xcode、Terminal、Simulator；「创意模式」换成 Figma、Pixelmator、Eagle。想改随时改，改完保存，下次直接调用。

**一键切换 / 快捷键切换**

切换方式很灵活：可以点菜单栏图标选方案，也可以自己设快捷键——对于手不爱离键盘的人来说这个很友好。另外还支持 Apple Shortcuts（快捷指令），可以跟 Focus 专注模式联动，进入专注模式时自动切换对应的 Dock 布局，这个组合用起来挺舒服的。

**添加多种项目**

Dock 里不只能放应用，DockFlow 还支持把文件夹、具体文件、甚至网址链接都放进去。需要快速打开某个项目文件夹？放进去，一点就到。

**添加空白间隔**

这个功能不起眼但挺实用。Dock 图标一堆挤在一起看着乱，DockFlow 支持在图标之间插入大/小两种宽度的空白，视觉上分组更清晰，找起来也快。

**配置导入/导出**

换了新 Mac？布局方案可以直接导出，在新机器上导入就能用，不用重新配置一遍。

**自动关闭非布局内应用**

切换方案时，可以选择让系统自动关掉不属于当前布局的应用。这个功能对真正想「切换工作状态」的人来说挺有用——不只是换换图标，连运行中的干扰项也一并清掉。

**锁定 Dock 到指定显示器**

多显示器用户应该懂那种痛：鼠标划过去，Dock 突然跑到另一块屏幕上去了。DockFlow 有 Lock Dock 功能，可以把 Dock 锁定在指定的显示器上，重启、睡眠唤醒之后依然有效。
